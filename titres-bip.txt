LISTE COMPLÈTE DES TITRES BIP
═══════════════════════════════════════════════════════

TRADUCTION
───────────────────────────────────────────────────────

"Comment dit-on en anglais ?"
→ Traduire un mot ou une phrase du français vers l'anglais
→ Exemple QUESTION : ici / c'est possible / nous sommes prêts

"Comment dit-on en français ?"
→ Traduire un mot ou une phrase de l'anglais vers le français
→ Exemple QUESTION : ready / it isn't going / my friend

"Que signifie ?"
→ Donner le sens d'un mot anglais
→ Exemple QUESTION : yet / so / like

═══════════════════════════════════════════════════════

PRONONCIATION
───────────────────────────────────────────────────────

"Comment se prononce ?"
→ Prononcer un mot dont la graphie est trompeuse
→ Exemple QUESTION : comfortable / terrible / possible

"Épelle le mot."
→ Donner les lettres du mot
→ Exemple QUESTION : they / ready / going

═══════════════════════════════════════════════════════

CONJUGAISON & FORMES
───────────────────────────────────────────────────────

"Quelle est la contraction de ?"
→ Contracter une forme longue
→ Exemple QUESTION : it is / we are / he is not / they are

"Développe la contraction."
→ Donner la forme longue d'une contraction
→ Exemple QUESTION : it's / we're / he isn't / I'm

"Mets à la forme négative."
→ Transformer une phrase affirmative en négative
→ Exemple QUESTION : he is ready / we are going / it's possible

"Mets à la forme interrogative."
→ Transformer une phrase en question
→ Exemple QUESTION : you are ready / he is going / it is possible

"Change le sujet."
→ Remplacer le sujet par celui indiqué
→ Exemple QUESTION : we are ready → avec "ils" / I'm going → avec "elle"

"Mets au pluriel."
→ Passer d'un sujet singulier au pluriel
→ Exemple QUESTION : he is ready / she is going

"Mets au singulier."
→ Passer d'un sujet pluriel au singulier
→ Exemple QUESTION : they are going / we are ready

═══════════════════════════════════════════════════════

CONSTRUCTION DE PHRASE
───────────────────────────────────────────────────────

"Complète la phrase."
→ Finir une phrase dont le début est donné par le prof
→ Exemple QUESTION : je suis... going / il n'est pas... ready / nous sommes...

"Quel est le mot manquant ?"
→ Trouver un mot précis dans une phrase incomplète
→ Exemple QUESTION : it ___ ready / we ___ not going / she ___

"Ajoute un mot."
→ Enrichir une phrase déjà connue avec un nouveau mot
→ Exemple QUESTION : it's comfortable → ajoute "ici"

"Construis la phrase."
→ Assembler plusieurs éléments donnés par le prof
→ Exemple QUESTION : elle / ne pas / aller / encore

═══════════════════════════════════════════════════════

SUBSTITUTION
───────────────────────────────────────────────────────

"Remplace par un pronom."
→ Substituer un nom par le pronom correspondant
→ Exemple QUESTION : my friend is going → remplace "my friend"

"Dis-le autrement."
→ Exprimer la même idée avec une autre structure
→ Exemple QUESTION : it's not ready / he is not = he isn't

"Remplace le mot."
→ Garder la structure, changer un mot précis
→ Exemple QUESTION : comfortable → terrible / ready → possible

═══════════════════════════════════════════════════════

RÉPÉTITION & ANCRAGE
───────────────────────────────────────────────────────

"Répète."
→ Reproduire exactement ce que le prof vient de dire
→ Exemple QUESTION : I'm going / we are not ready yet

"Encore une fois."
→ Répéter une structure déjà vue pour ancrer
→ Exemple QUESTION : il va / ça ne va pas
