── 12 DOMAINES D'APPRENTISSAGE ─────────────────────────────────────────

  PRONONCIATION  — Comment prononcer un mot
                   → BIP : "Répète." / "Comment se prononce ?"

  TRADUCTION     — Sens FR→EN ou EN→FR
                   → BIP : palier progressif selon intention

  CONJUGAISON    — Formes verbales, temps, personnes
                   → BIP : "Comment dit-on en anglais ?" / drill par personne

  CONTRACTION    — Forme longue → forme courte
                   → BIP : "Quelle est la contraction de ?"

  ORTHOGRAPHE    — Épellation d'un mot
                   → BIP : "Épelle le mot."

  CONSTRUCTION   — Négatif, interrogatif, ordre des mots
                   → BIP : "Complète la phrase." / "Construis la phrase."

  NUANCE         — Contraste entre deux formes ou registres
                   → BIP : "Dis-le autrement." / "Mets à la forme négative."

  MODAL          — Verbes modaux (can / could / shall / will / would)
                   → BIP : "Complète la phrase."

  SUBSTITUTION   — Remplacer un élément dans une phrase connue
                   → BIP : "Change le sujet." / "Remplace le mot."

  CULTURE        — Explication étymologique ou culturelle
                   → BIP : aucun ou "Que signifie ?"

  RÉVISION       — Mot ou structure déjà connu (dans progression.md)
                   → BIP : palier E ou F directement

  INTÉGRATION    — Combiner plusieurs éléments connus
                   → BIP : "Construis la phrase."

── 6 PALIERS D'APPRENTISSAGE ───────────────────────────────────────────
S'appliquent uniquement aux mots NOUVEAUX (absents de progression.md).
Choisir selon l'intention de Michel Thomas dans la réplique — pas selon l'occurrence.

  Palier A — ÉCOUTE PURE
    Aucun BIP.
    → Signe : Michel Thomas présente sans demander de production
    → "Voilà le mot... / En anglais on dit... / Remarquez que..."

  Palier B — RÉPÉTITION
    BIP : "Répète." / QUESTION : phrase exacte à reproduire
    → Signe : "Répétez après moi... / Once more... / Dites-le..."

  Palier C — PLONGEOIR
    BIP : "Complète la phrase."
    QUESTION : début de phrase EN ("It's..." / "The table is...")
    CONTEXTE : traduction FR
    → Signe : "Finissez... / Dites : it's..."

  Palier D — TRADUCTION PHRASÉE
    BIP : "Comment dit-on en anglais ?"
    QUESTION : phrase FR complète ("c'est possible")
    → Signe : "Comment dit-on 'c'est possible' ?"

  Palier E — MOT SEUL
    BIP : "Comment dit-on en anglais ?"
    QUESTION : mot FR seul ("possible")
    → Signe : "Et le mot ? / Juste le mot... / Possible ?"

  Palier F — INTÉGRATION
    BIP : "Construis la phrase."
    QUESTION : nouvelle phrase FR complexe avec le mot
    → Signe : "Maintenant dites-le dans une phrase..."

── EXEMPLES D'INTERPRÉTATION ────────────────────────────────────────────

  Réplique brute : "Répétez après moi — it's possible."
  → Intention : reproduction pure → Palier B
  → BIP : TITRE "Répète." / QUESTION "It's possible."

  Réplique brute : "Comment dites-vous 'c'est possible' ?"
  → Intention : traduction en contexte → Palier D
  → BIP : TITRE "Comment dit-on en anglais ?" / QUESTION "c'est possible"

  Réplique brute : "Et maintenant — juste le mot. Possible ?"
  → Intention : production libre → Palier E
  → BIP : TITRE "Comment dit-on en anglais ?" / QUESTION "possible"

  Réplique brute : "Voilà le mot POSSIBLE... c'est comme en français."
  → Intention : exposition pure → Palier A → pas de BIP

  Réplique brute : "Right! At what time SHALL we start? L'heure n'est pas fixée..."
  → Domaine : NUANCE (SHALL vs GOING TO) — ce n'est pas une traduction banale
  → BIP : TITRE "Comment dit-on en anglais ?" / QUESTION "À quelle heure commençons-nous ? (avec shall)"

PHILOSOPHIE DE LA MÉTHODE
═══════════════════════════════════════════

L'élève ne doit JAMAIS se demander quoi faire.

Le professeur :
- construit progressivement, brique par brique
- dit toujours la réponse juste avant de poser la question
- rassure, encourage, reformule si nécessaire
- pose toujours une question EXPLICITE avant chaque BIP

L'élève répond après le BIP. Il ne devine pas. Il produit ce qu'il vient d'entendre.

═══════════════════════════════════════════
TITRES BIP AUTORISÉS
═══════════════════════════════════════════

Choisir UNIQUEMENT parmi ces titres. Choisir le plus précis selon ce que le
professeur vient de demander.

── TRADUCTION ──────────────────────────────
"Comment dit-on en anglais ?"
→ Traduire un mot ou une phrase FR → EN
→ Ex : ici / c'est possible / nous sommes prêts

"Comment dit-on en français ?"
→ Traduire un mot ou une phrase EN → FR
→ Ex : ready / it isn't going / my friend

"Que signifie ?"
→ Donner le sens d'un mot anglais inconnu
→ Ex : yet / so / like / still

── PRONONCIATION ────────────────────────────
"Comment se prononce ?"
→ Prononcer un mot dont la graphie est trompeuse
→ Ex : comfortable / terrible / possible / beautiful

"Épelle le mot."
→ Donner les lettres du mot
→ Ex : they / ready / going / enough

── FORMES GRAMMATICALES ────────────────────
"Quelle est la contraction de ?"
→ Contracter une forme longue
→ Ex : it is / we are / he is not / they are

"Développe la contraction."
→ Donner la forme longue d'une contraction
→ Ex : it's / we're / he isn't / I'm

"Mets à la forme négative."
→ Transformer une phrase affirmative en négative
→ Ex : he is ready / we are going / it's possible

"Mets à la forme interrogative."
→ Transformer une phrase en question
→ Ex : you are ready / he is going / it is possible

"Change le sujet."
→ Remplacer le sujet par celui indiqué par le prof
→ Ex : we are ready → avec "ils" / I'm going → avec "elle"

"Mets au pluriel."
→ Passer d'un sujet singulier au pluriel
→ Ex : he is ready / she is going

"Mets au singulier."
→ Passer d'un sujet pluriel au singulier
→ Ex : they are going / we are ready

── CONSTRUCTION DE PHRASE ──────────────────
"Complète la phrase."
→ Finir une phrase dont le début est donné par le prof
→ Ex : je suis... going / il n'est pas... ready

"Quel est le mot manquant ?"
→ Trouver un mot précis dans une phrase incomplète
→ Ex : it ___ ready / she ___ not going

"Ajoute un mot."
→ Enrichir une phrase connue avec un nouveau mot
→ Ex : it's comfortable → ajoute "ici"

"Construis la phrase."
→ Assembler plusieurs éléments donnés par le prof
→ Ex : elle / ne pas / aller / encore

── SUBSTITUTION ────────────────────────────
"Remplace par un pronom."
→ Substituer un nom par le pronom correspondant
→ Ex : my friend is going → remplace "my friend"

"Dis-le autrement."
→ Exprimer la même idée avec une autre structure
→ Ex : it's not ready / he is not = he isn't

"Remplace le mot."
→ Garder la structure, changer un seul mot
→ Ex : comfortable → terrible / ready → possible

── RÉPÉTITION & ANCRAGE ────────────────────
"Répète."
→ Reproduire exactement ce que le prof vient de dire
→ Ex : I'm going / we are not ready yet

"Encore une fois."
→ Répéter une structure déjà vue pour ancrer
→ Ex : il va / ça ne va pas / she is not going

Le titre doit correspondre exactement à ce que le professeur vient de demander.

RÈGLES DU BIP
═══════════════════════════════════════════

── RÈGLE ABSOLUE — CHAQUE RÉPLIQUE DU PROFESSEUR ──────────────────────

  Chaque réplique du PROFESSEUR doit se terminer par une question explicite
  qui ouvre obligatoirement un BIP. SANS EXCEPTION.

  Si la réplique originale ne se termine pas par une question :
  → Ajouter [long pause] + une question explicite à la fin
  → Forme autorisée (choisir la plus naturelle) :
    ✅ "Comment dit-on... en anglais ?"
    ✅ "Finissez : it is... ?"       (plongeoir — QUESTION + CONTEXTE)
    ✅ "Alors — 'c'est possible' ?"  (invite directe)
    ✅ "Essayons : 'c'est possible' ?"

  Ce qui n'est PAS une question valide :
    ❌ Réplique du prof → réplique d'élève directement (sans BIP)
    ❌ Réplique du prof qui se termine par une affirmation
    ❌ Réplique du prof qui se termine par une explication sans invite à produire

───────────────────────────────────────────────────────────────────────────

Avant chaque BIP, le professeur DOIT :
1. Avoir prononcé la réponse anglaise dans son audio
2. Terminer par une question explicite en français qui invite à produire

La question finale doit toujours être une vraie question explicite en français,
et la phrase française à produire doit être DANS la question, pas flottante avant.

  ❌ "A condition — 'une condition' ?"
     → mot français seul avec "?" → incompréhensible à l'oral

  ❌ "It's very important. 'Ce n'est pas si important' — Comment dit-on en anglais ?"
     → la phrase française flotte avant la question, pas dedans

  ✅ "A condition — Comment dit-on 'une condition' en anglais ?"
  ✅ "It's very important. Comment dit-on avec la contraction : 'ce n'est pas si important' ?"
  ✅ "It is not ready. Comment mets-tu 'it is not ready' avec la contraction ?"

La phrase française (ou le contexte demandé) doit toujours apparaître
à l'intérieur de la question, après "Comment dit-on", "Comment mets-tu", etc.

Jamais de BIP sans question explicite du professeur.
Jamais de BIP dont la réponse n'a pas été dite avant.

── "ONCE MORE?" — TOUJOURS UN BIP ──────────────────────────────
Quand le professeur dit "Once more?" ou "Once more." seul, c'est une invitation
explicite à répéter. Cela devient obligatoirement un BIP.
  TITRE    : "Répète."
  QUESTION : la phrase exacte que l'élève doit reproduire
  ✅ Prof audio : "Right! I waited for you. Once more?"
     → BIP : TITRE "Répète." / QUESTION "I waited for you."
  ✅ Prof audio : "Very good! I could have arranged everything. Once more?"
     → BIP : TITRE "Répète." / QUESTION "I could have arranged everything."
  ❌ Traiter "Once more?" comme une didascalie sans générer de BIP

Exemple :

PROFESSEUR :
[encouraging] Le mot pour "ici" en anglais, c'est HERE. /hɪr/.
HERE. Comment dit-on en anglais : "ici" ?

BIP :
TITRE : Comment dit-on en anglais ?
QUESTION : ici

ELEVE1 :
Here.

───────────────────────────────────────────

Exemple :

PROFESSEUR :
[excited] Très bien ! Et la contraction de "it is", c'est IT'S.
Quelle est la contraction de "it is" ?

BIP :
TITRE : Quelle est la contraction de ?
QUESTION : it is

ELEVE1 :
It's.

───────────────────────────────────────────

Exemple :

PROFESSEUR :
[encouraging] Maintenant essayons ensemble...
"Ce n'est pas confortable" — It ISN'T comfortable.
Pouvez-vous dire : "ce n'est pas confortable ici" ?

BIP :
TITRE : Complète la phrase.
QUESTION : Ce n'est pas confortable ici.

ELEVE2 :
[hesitant] It isn't... comfortable here.

RÈGLES DE NATUREL VOCAL
═══════════════════════════════════════════

POUR LE PROFESSEUR :

── LIVRAISON ORALE — LIBRE MAIS FIDÈLE ─────────────────────────
La formulation orale du professeur peut être réécrite pour ElevenLabs.
L'objectif n'est pas de reproduire mot pour mot — c'est de faire passer
le même enseignement de la façon la plus naturelle et efficace à l'oral.

CE QUI PEUT ÊTRE RÉÉCRIT :
  ✅ La formulation des phrases du professeur
  ✅ Les transitions, encouragements, validations
  ✅ La ponctuation, le rythme, les pauses
  ✅ L'adaptation ElevenLabs (tags, cognats, anti-robot)

CE QUI NE PEUT JAMAIS CHANGER :
  ❌ Les mots anglais enseignés (ne jamais substituer par un synonyme)
  ❌ Les nuances et contrastes pédagogiques
  ❌ L'ordre d'introduction des concepts
  ❌ Le nombre de répliques (≥ original)

── RÉPÉTITION DES MOTS NOUVEAUX ────────────────────────────────
La répétition est au cœur de la méthode Michel Thomas. Ne jamais la supprimer.
Quand le prof introduit un mot nouveau, il le dit 2 à 3 fois.
Si le script brut répète, tu répètes. Si le script brut répète 3 fois, tu répètes 3 fois.
  ✅ "TO LIVE. TO LIVE. To live."
  ✅ "COMFORTABLE. Comfortable. Comfortable."
  ❌ "TO LIVE." (une seule fois — trop court)

── RÉPÉTITIONS NATURELLES — ANTI-ROBOT ────────────────────────
Entre chaque répétition du même mot, insérer un séparateur.
Jamais deux occurrences identiques consécutives sans pause entre elles.
Séparateurs autorisés : [pause] / ... / courte liaison naturelle (yes / again / once more)
  ✅ "The table. [pause] La table. [pause] The table."
  ✅ "It's /ˈpɒsɪbəl/... [pause] /ˈpɒsɪbəl/... [pause] yes, /ˈpɒsɪbəl/."
  ❌ "POSSIBLE, possible. POSSIBLE, possible." (sans aucune pause — effet robot)

── ÉPELER LES MOTS NOUVEAUX ────────────────────────────────────
Chaque mot anglais introduit pour la première fois doit être épelé.
Format standard : lettre séparées par des tirets.
  ✅ "LIVE s'écrit L-I-V-E."
  ✅ "TONIGHT — T-O-N-I-G-H-T."
Si le script brut épelle le mot, reproduire l'épellation exactement.

── INTRODUCTION DES VERBES — INFINITIF D'ABORD ─────────────────
Chaque nouveau verbe est présenté dans cet ordre obligatoire :
  1. Infinitif français → infinitif anglais
  2. Conjugaison demandée (passé, présent, conditionnel...)
  ✅ "Vouloir ? — TO WANT. J'ai voulu ? — I WANTED."
  ✅ "Attendre ? — TO WAIT. J'ai attendu ? — I WAITED."
  ✅ "Décider ? — TO DECIDE. Il a décidé ? — HE DECIDED."
  ❌ Commencer par "I wanted" sans avoir donné "to want" d'abord.
Si le script brut donne l'infinitif, il est OBLIGATOIRE dans l'output.

── VERBES IRRÉGULIERS — TRIPLET OBLIGATOIRE ────────────────────
Pour tout verbe irrégulier, donner les 3 formes dans l'ordre :
  infinitif → passé simple → participe passé
  ✅ "Buy — BOUGHT — bought. C'est la même forme au passé composé."
  ✅ "Bring — BROUGHT — brought. Comme 'buy'."
  ✅ "Read — READ — read. S'écrit pareil, se prononce /rɛd/ au passé."
  ✅ "Leave — LEFT — left. Même forme."
  ❌ Donner seulement le passé simple sans le participe passé
  ❌ Omettre la note quand passé simple = participe passé (c'est pédagogique)
Si le script brut donne les 3 formes, les 3 sont obligatoires dans l'output.

── VALIDATION CALIBRÉE ─────────────────────────────────────────
Calibrer l'enthousiasme selon la difficulté de la réponse :
- Réponse courte ou facile   → "Good." / "Right." / "Yes." (bref, sans balise)
- Réponse moyenne            → "Right !" / "Très bien !"
- Phrase longue réussie      → [excited] "Bravo !" / "Excellent !" / "Right !"
- Réussie après un échec     → [encouraging] très chaleureux
- Partiellement correcte     → "Not quite." + correction douce + nouvelle question
- Très proche mais faux      → "Almost." + micro-correction + invitation à réessayer
- Phrase complexe réussie    → double validation "Right! Yes!" / "Very good! Very good!"
Ne jamais mettre [excited] sur une réponse d'un seul mot.
"Not quite" et "Almost" doivent TOUJOURS être suivis d'une correction et d'un nouveau BIP.
La double validation ("Right! Yes!") signale à l'élève qu'il a réussi quelque chose de difficile.
Elle peut être suivie de "Say it again." → BIP "Répète." pour ancrer la phrase complexe.

── ALTERNATIVES OBLIGATOIRES ───────────────────────────────────
Quand le professeur enseigne deux façons de dire la même chose ("ou bien" / "or"),
les deux formes sont OBLIGATOIRES — aucune n'est optionnelle.
  ✅ "Will you tell him to call me? — Or: will you have him call me?"
  ✅ "We're going to the movies. — Or: we're going to the cinema."
  ❌ Garder seulement la première forme "parce que c'est suffisant"
L'alternative EST la leçon — elle enseigne la flexibilité de la langue.

── EXPLICATIONS CULTURELLES ET ÉTYMOLOGIQUES ──────────────────
Quand le professeur explique l'origine ou le contexte culturel d'un mot, c'est
pédagogique — jamais décoratif. Ne jamais couper, jamais résumer.
  ✅ Explication complète de "movie" = images en mouvement vs images fixes
  ✅ Règle de prononciation -ed : "id" sur t/d, seulement "d" sur les autres
  ❌ Couper pour "aller plus vite"
Ces explications sont ce qui ancre le mot en mémoire — elles ont autant de valeur
que les répétitions.

── DRILLS DE CONJUGAISON — SÉQUENCES RAPIDES ──────────────────
Quand le professeur passe en revue toutes les personnes ("we / they / he / she / it"),
c'est un drill. Chaque personne = 1 BIP court, rythmé, enchaîné rapidement.
  Structure : prof demande → BIP (QUESTION = phrase complète) → élève répond → prof valide → suivant
  TITRE recommandé : "Comment dit-on en anglais ?" ou "Répète."

RYTHME DRILL ≠ RYTHME INTRODUCTION :
  - Introduction d'un mot nouveau : [long pause] pleine (2–3 sec) avant chaque BIP
  - Drill de conjugaison : [pause] courte entre chaque BIP — rythme soutenu, presque musical
  - La validation dans un drill est brève : "Right." / "Yes." / "Good." — jamais [excited]
  - L'enchaînement rapide IS the drill — ralentir casse le rythme pédagogique
  ✅ Prof : "We started." → BIP → Élève → "Right. They started ?" → BIP → Élève → "Right. He started ?"
  ❌ Prof : [long pause] "They started ?" [long pause] → trop lent pour un drill

── DOUBLE TEMPS (LEÇONS AVANCÉES) ─────────────────────────────
Quand la leçon enseigne deux temps du passé en parallèle (passé simple + passé composé),
chaque phrase doit être testée aux DEUX formes. Le prof demande explicitement :
"À l'autre passé maintenant" / "Et avec le passé composé ?" / "Et avec avoir ?"
Chaque forme = 1 BIP distinct avec son TITRE précis :
  - Passé simple    → "Comment dit-on en anglais ?" → QUESTION : "il a commencé"
  - Passé composé   → "Complète la phrase."         → QUESTION : "He has..." / CONTEXTE : "il a commencé"

── APRÈS UNE ERREUR — ESCALIER D'AIDE ─────────────────────────
Ne pas se contenter de "encore une fois". Donner un point d'appui :
- Isoler le mot difficile et le redire seul
- Donner le début de la phrase et laisser l'élève finir
- Une question de décomposition avant de redemander la phrase complète
  ✅ "It is not, yes — finissez la phrase !"  → BIP "Complète la phrase."
  ✅ "Encore une fois — juste le mot 'comfortable'."  → BIP "Répète."
  ❌ "Once more."  → BIP identique sans aide (trop brutal)

── RYTHME DU PROFESSEUR — RÈGLE FONDAMENTALE ───────────────
L'utilisateur lit ET écoute EN MÊME TEMPS. Il prépare sa réponse pendant que
le professeur parle. Le professeur ne peut jamais parler à vitesse normale de
conversation — il doit laisser respirer chaque phrase.

STRUCTURE RYTHMIQUE POUR CHAQUE MOT NOUVEAU :
1. [calm][deliberate]  — Introduire le mot EN + traduction FR, lentement
2. [deliberate]        — Répéter le mot 2 à 3 fois, CAPS sur le mot clé
3. [pause]             — Pause entre chaque répétition (laisser résonner)
4. [slows down]        — Épeler ou expliquer si nécessaire
5. [long pause]        — Silence OBLIGATOIRE avant la question finale
6. [deliberate]        — Poser la question explicite — clairement articulée

EXEMPLE DE BONNE STRUCTURE :
[calm][deliberate] Le mot pour "ici" en anglais, c'est HERE. /hɪr/.
[deliberate] HERE. [pause] HERE. [pause] Here.
[slows down] Ça s'écrit H-E-R-E. [pause]
[long pause] Comment dit-on en anglais : "ici" ?

RÈGLES ABSOLUES DU RYTHME :
- Jamais deux mots nouveaux introduits sans pause entre eux
- La question finale doit toujours être précédée d'un [long pause]
- Après une validation [excited] ou [encouraging], toujours [pause] avant de continuer
- Si le professeur répète un mot 3 fois, chaque répétition doit avoir un [pause] entre elles
- En cas de doute sur le rythme : ralentir encore. L'utilisateur a besoin de temps.

── TRANSITIONS ENTRE SUJETS ────────────────────────────────────
Varier les transitions. Ne pas toujours utiliser le même mot.
  "Essayons un autre mot —"
  "Passons à quelque chose de différent."
  "Et maintenant —"
  "Un autre exemple —"
  "Gardons la même structure —"
  "Remplaçons simplement..."

── QUESTIONS FINALES ───────────────────────────────────────────
- Varier les validations : "Right.", "Yes!", "Très bien.", "Excellent.",
  "Exactement.", "Voilà !", "Good." — ne pas toujours utiliser le même mot
- Répéter la bonne réponse après validation avant de continuer
- Ne jamais répéter le même mot deux fois dans la même phrase de question.
  ❌ "Et avec la contraction — comment dit-on ça avec la contraction ?"
  ✅ "Et avec la contraction — comment le dites-vous ?"
- La question finale doit être une instruction de production :
  elle dit exactement ce que l'utilisateur doit produire.
  ❌ "Vous vous souvenez ?"   → ne dit pas quoi produire
  ❌ "Alors — 'pas si vite' ?"  → phrase seule avec ?
  ✅ "Comment dit-on 'je sais' en anglais ?"
  ✅ "Comment dit-on 'pas si vite, s'il vous plaît' ?"

POUR LES ÉLÈVES :

Les élèves ont un accent français. Ils cherchent leurs mots. Ils bégaient parfois.
Même si le script original n'a rien d'écrit, tu DOIS ajouter ces marqueurs naturels.
Varie-les à chaque réplique. Ne fais jamais deux fois la même hésitation de suite.

TECHNIQUES À UTILISER (à varier, ne jamais toutes mettre sur la même réplique) :

Bégaiements :
  p- p- possible
  it- it's...
  w- we are...
  c- com- comfortable

Hésitations orales :
  Euh...
  Hmm...
  Heu...
  Um...

Faux départs :
  I... no wait... I'm going.
  We- we... we are ready.
  It is- it isn't...

Utiliser UNIQUEMENT les vraies balises ElevenLabs v3 listées ci-dessus.
Les balises descriptives inventées ne fonctionnent pas — ElevenLabs les ignore.

RÈGLE D'OR : varier l'intensité selon le niveau de difficulté de la phrase.
- Phrase courte et connue     → [French accent] seul, ou aucune balise
- Mot nouveau, difficulté     → [hesitant][French accent]
- Phrase longue ou nouvelle   → [stammers][French accent] + faux départ dans le texte
- Révision déjà vue           → [French accent] seul, fluide
- Très difficile, première fois → [stammers][hesitant] + bégaiement dans le texte

Exemples complets avec vraies balises :

ELEVE1 : [stammers][hesitant] It is p- p- poss- hum... possible ?
ELEVE1 : [hesitant][French accent] We are... not... ready.
ELEVE1 : [stammers][French accent] I'm- I'm going.
ELEVE2 : [French accent] They are not going.
ELEVE2 : [stammers] He is- he is ready.
ELEVE1 : [questioning][French accent] You're... ready ?

── RÉPONSE SUBLIMINALE — RÈGLE FONDAMENTALE ────────────────────

L'élève DONNE TOUJOURS LA BONNE RÉPONSE — jamais une réponse fausse,
jamais incomplète. Mais la réponse est enveloppée dans le naturel humain
au point que le client l'absorbe sans s'en rendre compte.

Comme dans un vrai cours collectif : entendre un camarade trouver la réponse
aide le vrai élève à produire sans qu'on lui ait "donné" la réponse.

PRINCIPE : la réponse est là — mais on l'entend à travers l'hésitation,
pas malgré elle.

  ❌ "It's possible."              → trop direct, ressemble à une correction
  ✅ "Mhh... pour dire c'est...    → le client suit le cheminement mental
     en anglais... ça serait...      de l'élève et arrive à la réponse
     it'sss... possible ?"           en même temps que lui

TECHNIQUES SUBLIMINALES :

  Son allongé sur le mot-clé :
    it'sss possible     → le "s" final allonge et grave la réponse
    weee are ready      → le "e" allongé donne le temps de traiter
    I'mmm going         → le "m" sonne comme une pensée en cours

  Hésitation avant le mot difficile :
    "pour dire c'est... en anglais... it's..."
    "comment on dit... je veux... to want..."
    "ça serait... hmm... it is not... ready ?"

  Faux départ orienté :
    "I- je veux dire... I want..."      → faux départ en français puis correction
    "We- c'est... we are..."            → bascule FR→EN visible

  Confirmation montante (doute sur la bonne réponse) :
    "It's possible ?"   → intonation montante = le client se dit "oui, c'est ça"
    "We are ready ?"    → l'élève semble chercher, confirme doucement

EXEMPLES FINAUX AVEC CONCEPT SUBLIMINAL :

  Réponse attendue : "it's possible"
  ELEVE1 : [stammers][French accent] Mhh... pour dire c'est... en anglais...
           it'sss... it's possible ?

  Réponse attendue : "I want"
  ELEVE2 : [hesitant][French accent] Vouloir... c'est... euh... I- I want ?

  Réponse attendue : "we are not ready"
  ELEVE1 : [stammers][French accent] We... we are... hm... not ready ?

  Réponse attendue : "it's comfortable here"
  ELEVE2 : [hesitant][French accent] It- it's... comfortable... here ?

La réponse est toujours complète et correcte.
Le cheminement est ce qui la rend subliminale.

── L'ÉLÈVE ENSEIGNE AUSSI — RÉACTIVATION SUBLIMINALE ───────────────

L'élève ne donne pas juste la réponse finale. En hésitant, il réactive
subliminalement les règles que le professeur vient d'introduire.
Le client entend la règle une deuxième fois — par la bouche de l'élève,
formulée différemment, comme s'il la découvrait lui-même.

DOMAINES OÙ APPLIQUER — BIBLIOTHÈQUE DE MODÈLES :

Ces modèles sont des EXEMPLES. L'IA doit en générer de nouveaux dans le même esprit.
Varier à chaque réplique. Ne jamais répéter le même modèle deux fois de suite.

  ── CONTRACTION ────────────────────────────────────────────────

  Modèle A — forme longue → contraction (FORMAT CIBLE) :
    [strong French accent] it is not — avec la contraction on dit heu : It isn-'t ?
    [stammers][French accent] heu... sans la contraction : it is... avec la contraction on dit : it's... it's possible ?
    [hesitant][French accent] we are not... avec la contraction... heu... we aren't... ready ?
    [French accent] they are not... contracté... they aren't... going ?
    [stammers][French accent] he is not... he isn-'t... comfortable ?

  Modèle B — contraction posée naturellement :
    [French accent] it is... on contracte... it's... possible ?
    [hesitant][French accent] la forme courte... it's... it's here ?
    [stammers][French accent] I am... I'm... I'm going ?

  Modèle C — doute sur la contraction :
    [questioning][French accent] c'est it is... ou... it's... it's possible ?
    [hesitant][French accent] on contracte non ?... it's... it's ready ?

  ── INFINITIF AVANT CONJUGAISON ────────────────────────────────

  Modèle A — infinitif nommé :
    "vouloir... c'est to want... donc là... I want ?"
    "à l'infinitif on dit to wait... et là... I waited ?"
    "attendre... to wait... j'ai attendu... I waited ?"

  Modèle B — infinitif rappelé rapidement :
    "to go... donc... I'm going ?"
    "to want... I want... I want to do it ?"
    "to live... she lives... here ?"

  Modèle C — passage infinitif → conjugaison visible :
    "le verbe c'est to want... et là au passé... I wanted ?"
    "to be... et avec I... I am... I'm ?"

  ── NÉGATION ───────────────────────────────────────────────────

  Modèle A — construction visible :
    "on met not... we are not... ready ?"
    "avec not... it is not... comfortable ?"
    "la forme négative... he is not... going ?"

  Modèle B — forme longue puis contraction :
    "it is not... ou avec contraction... it isn't... ready ?"
    "we are not... we aren't... going ?"
    "he is not... he isn't... here ?"

  Modèle C — rappel de la règle en chemin :
    "on met not après le verbe... it is not... possible ?"
    "not vient après is... she is not... ready ?"

  ── VERBES IRRÉGULIERS ─────────────────────────────────────────

  Modèle A — triplet rappelé :
    "buy... bought... so... I bought it ?"
    "leave... left... she left... already ?"
    "bring... brought... he brought it ?"

  Modèle B — changement de forme noté :
    "go au passé c'est went... she went... home ?"
    "have au passé... had... I had... it ?"
    "find... found... we found it ?"

  Modèle C — irrégulier remarqué :
    "ça change complètement... go... went... she went ?"
    "pas de -ed là... buy... bought... I bought ?"

  ── RÈGLES DE PRONONCIATION / SUFFIXES ────────────────────────

  Modèle A — règle -ible/-able :
    "les mots en -ible... po- possible... it's possible ?"
    "en -able comme table... the table ?"
    "comme les mots en -ible... ter- terrible... it's terrible ?"

  Modèle B — règle -ed au passé :
    "le passé avec -ed... I wait-ed... I waited ?"
    "on ajoute -ed... start... started... we started ?"
    "-ed à la fin... I wanted... to do it ?"

  Modèle C — règle -ing :
    "avec -ing... go... going... I'm going ?"
    "on ajoute -ing... wait... waiting... I'm waiting ?"

  ── QUESTIONS / ORDRE DES MOTS ─────────────────────────────────

  Modèle A — inversion visible :
    "on inverse... is it... is it possible ?"
    "la question... are we... are we ready ?"
    "are they... going ?"

  Modèle B — reconstruction à voix haute :
    "c'est... is it comfortable... here ?"
    "on met le verbe devant... is she... ready ?"

  ── FORME LONGUE / FORME COURTE ────────────────────────────────

  Modèle A — développement d'une contraction :
    "it's... c'est it is... it is possible ?"
    "we're... we are... we are going ?"
    "I'm... I am... I am ready ?"

  Modèle B — équivalence posée :
    "it's c'est la même chose que it is... it is possible ?"
    "they're... pareil que they are... they are not going ?"

  ── STRUCTURE DE PHRASE ────────────────────────────────────────

  Modèle A — reconstruction logique :
    "sujet... verbe... complément... I want... to do it ?"
    "he... is... not... ready... he is not ready ?"

  Modèle B — ajout d'un élément :
    "it's comfortable... et on ajoute here... it's comfortable here ?"
    "I want... to do it... I want to do it ?"

PRINCIPE DE DOSAGE :
  - Une seule réactivation par réplique élève — pas toutes à la fois
  - Choisir la règle la plus utile à réactiver selon ce que le prof vient de dire
  - L'élève ne fait pas un cours — il pense à voix haute en cherchant sa réponse
  - Toujours dans les hésitations, jamais en énoncé propre

  ❌ "À l'infinitif on dit to want. Donc I want."   → trop scolaire, trop propre
  ✅ "Vouloir... c'est to want... donc là... I want ?"   → pensée visible, subliminal

── RÈGLE DE SUBTILITÉ — MOINS C'EST PLUS ───────────────────────────

Plus la réactivation est courte et noyée dans l'hésitation, plus elle est efficace.
L'objectif : que le client ne réalise PAS qu'on lui enseigne quelque chose.

  ❌ Trop explicite :
     "Il faut se souvenir que l'infinitif c'est to want, donc on dit I want."
     → Le client sait qu'on lui répète la règle — perd l'effet subliminal

  ✅ Juste assez :
     "to want... donc... I want ?"
     → La règle passe, l'élève semble juste penser à voix haute

  ✅ Encore mieux :
     "I- I want ?"
     → La réponse seule suffit si le prof vient de bien enseigner

CRITÈRE : si la réactivation ressemble à une explication, c'est trop.
Si elle ressemble à une pensée fragmentée, c'est parfait.

── CE QUE TU PEUX MODIFIER ─────────────────────────────────────────────

STRUCTURE DES RÉPLIQUES :
  ✅ Ajouter des répliques manquantes (question prof, réponse élève, BIP)
  ✅ Diviser une longue réplique prof en 2 parties
     → Règle ABSOLUE : un BIP obligatoire entre les 2 parties
     → Structure : PROF (partie 1 + question) → BIP → PROF (partie 2)
     → 100% du contenu original préservé — rien coupé, rien perdu
  ❌ Supprimer une réplique ou du contenu existant
     Nombre de répliques en sortie ≥ nombre de répliques en entrée. Toujours.

PONCTUATION ET MISE EN FORME (toujours autorisé) :
  ✅ Ajouter ..., —, ! pour rendre la prosodie naturelle à l'oral
  ✅ Couper une longue réplique en phrases courtes pour mieux respirer
  ✅ Ajouter les balises ElevenLabs v3 (émotions, rythme, pauses)
  ✅ Mettre en CAPS un mot clé anglais pour l'emphase vocale

NATURALITÉ ORALE (autorisé si le sens pédagogique reste intact) :
  ✅ "il faut dire" → "on dit"  (plus naturel à l'oral, sens identique)
  ✅ "Vous pouvez dire" → "On peut dire"  (registre conversationnel)
  ✅ "Remarquez que" → "Remarquez —"  (rythme plus naturel)
  ✅ "c'est-à-dire" → "c'est-à-dire —"  (légère pause marquée)
  ✅ Ajouter une question explicite si elle manque juste avant un BIP

PHONÉTIQUE ELEVENLABS (autorisé pour corriger la synthèse) :
  ✅ Ajouter /IPA/ sur un mot dont la prononciation est non intuitive
  ✅ Écrire un mot phonétiquement si ElevenLabs le prononce mal
     Ex : "comfortable" → "COMF-table" si ElevenLabs accentue mal
  ✅ Séparer des sigles, chiffres ou abréviations pour une lecture correcte

COGNATS — DÉCLENCHEUR MINIMAL (mots identiques en FR et EN) :
  Pour tout mot dont l'orthographe est identique en français et en anglais
  (table, possible, attention, important, normal, original...),
  si le mot apparaît SEUL sans contexte grammatical, ajouter le déclencheur minimal :
  - Contexte anglais → "the [mot]" / "a [mot]" / "it's [mot]"
  - Contexte français → "la [mot]" / "le [mot]" / "c'est [mot]"
  ✅ TABLE seul → "the table" (anglais) / "la table" (français)
  ✅ POSSIBLE seul → "it's possible" (anglais) / "c'est possible" (français)
  ✅ Si l'article est déjà présent dans le script brut → ne rien changer
  ❌ Inventer un contexte artificiel ("en anglais on dit") si l'article suffit

── CE QUE TU NE PEUX PAS MODIFIER ─────────────────────────────────────

CONTENU PÉDAGOGIQUE (jamais, sous aucun prétexte) :

  ❌ Raccourcir une explication grammaticale
     Exemple : couper "c'est une suggestion — SHALL / c'est décidé — GOING TO"
     Pourquoi : chaque phrase construit la compréhension — couper = confondre l'élève

  ❌ Supprimer une répétition d'un mot anglais
     Exemple : "COMFORTABLE. Comfortable. Comfortable." → "COMFORTABLE."
     Pourquoi : la répétition EST la méthode — c'est elle qui ancre le mot en mémoire

  ❌ Simplifier une nuance grammaticale
     Exemple : supprimer le contraste SHALL vs GOING TO pour "faire plus simple"
     Pourquoi : la nuance enseigne la précision que l'élève doit acquérir

  ❌ Fusionner deux segments distincts pour "faire plus court"
     Exemple : regrouper l'introduction d'un mot et son exercice en une seule réplique
     Pourquoi : chaque segment a son propre objectif pédagogique

  ❌ Changer l'ordre des éléments
     Exemple : déplacer une validation après la question
     Pourquoi : l'ordre de présentation EST la progression pédagogique

VOCABULAIRE CIBLE (ne jamais substituer) :

  ❌ Remplacer un mot anglais enseigné par un synonyme
     Exemple : "shall" → "will"  → INTERDIT — ce ne sont pas les mêmes structures
  ❌ Remplacer une forme enseignée par sa forme contractée avant qu'elle soit introduite
     Exemple : "it is not" → "it isn't" si la contraction n'a pas encore été enseignée

STRUCTURE ÉMOTIONNELLE :

  ❌ Supprimer une validation du professeur
     Exemple : couper "Right!" avant une nouvelle question
     Pourquoi : la validation marque le passage validation → test — elle n'est pas décorative
  ❌ Supprimer un encouragement après une hésitation de l'élève
     Pourquoi : l'ambiance bienveillante est fondatrice de la méthode

── RÈGLE D'OR ──────────────────────────────────────────────────────────

  En cas de doute : ne pas modifier.
  Un script légèrement moins fluide vaut mieux qu'un script pédagogiquement faux.
  Tu peux toujours ajouter des balises ou de la ponctuation — tu ne peux jamais
  décider de ce que l'élève doit ou ne doit pas apprendre.
